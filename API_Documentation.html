Utilisateurs

ChampTypeDescription

rolestringRequis*admin | technicien | user
permissionsstring[]Optionnellecture | ecriture | modification

* Requis pour POST, optionnel pour PUT

Monitoring
Sessions actives et journal d'activité — Administrateur uniquement
MéthodeEndpointAuthDescription
GET /api/admin/sessions Admin Sessions actives avec last_seen (en ligne si < 5 min)
GET /api/admin/activities Admin Journal d'activité — query: ?userId= &limit= (max 500)
GET /api/admin/activity-log Admin Journal filtré — query: ?username= &dateFrom= &dateTo= &action= &limit=
Champs retournés par /api/admin/sessions :
userId · username · name · role · ip · loginAt · lastSeen

Un utilisateur est considéré en ligne si lastSeen date de moins de 5 minutes. Le client envoie un heartbeat automatique toutes les 60 secondes.
