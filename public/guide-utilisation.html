1. Introduction

Gestion Équipements IT est une application web complète de gestion de parc informatique. Elle permet de :

  • Gérer l'ensemble des équipements (ordinateurs, serveurs, réseaux, imprimantes, etc.)
  • Suivre les maintenances, tickets d'assistance et interventions
  • Gérer les garanties, licences logicielles et contrats de maintenance
  • Planifier et suivre les visites de site
  • Gérer les transferts d'équipements entre sites
  • Créer et approuver des demandes d'achat
  • Gérer les retours garantie (RMA)
  • Détecter automatiquement les anomalies
  • Générer des rapports détaillés et exporter en Excel/PDF
  • Communiquer via la messagerie intégrée
Note : Les fonctionnalités accessibles dépendent de votre rôle et de vos permissions. Consultez la section Rôles & Permissions pour plus de détails.

2. Connexion & Déconnexion

2.1 Page de connexion

  • Ouvrez l'application dans votre navigateur
  • Saisissez votre nom d'utilisateur et mot de passe
  • Cliquez sur Connexion

2.2 Session

  • La session expire après 5 minutes d'inactivité
  • Un avertissement s'affiche avant l'expiration — cliquez sur Rester connecté pour prolonger
  • Cliquez sur votre avatar en haut à droite puis Déconnexion pour quitter
Attention : Après 5 tentatives de connexion échouées, votre compte est temporairement bloqué pendant 15 minutes.

3. Tableau de bord

Le tableau de bord est la page principale après connexion. Il affiche :

3.1 Barre de navigation supérieure

  • Marque : « Gestion IT » (clic = retour au dashboard)
  • Boutons de navigation (défilables) : Équipements, Transferts, Assistance, Maintenance, Visites, Garanties, Renouv. Garantie, Rapports, Calendrier, Carte, Inventaire, TV Dashboard
  • Boutons administrateur : Journal, Monitoring, Utilisateurs, Sites, Fournisseurs, Sauvegarde
  • Barre d'actions droite : Recherche (Ctrl+K), Messagerie, Notifications, Ping All, Masquer données sensibles, Scanner QR, Mode sombre, Profil utilisateur, Déconnexion

3.2 Indicateurs KPI

6 cartes cliquables affichant :

  • Total équipements — clic = afficher tout le parc
  • Actifs — clic = filtrer les équipements actifs
  • Défaillants — clic = filtrer les équipements défaillants
  • Tickets ouverts — clic = ouvrir le module Maintenance
  • Non visités — clic = afficher les équipements sans visite
  • Visites planifiées — clic = ouvrir le module Visites

3.3 Graphiques

  • Donut : Répartition des équipements par statut (Actif, Maintenance, Défaillant, Inactif, Réformé)
  • Barres : Répartition par type (Ordinateur, Réseau, Serveur, Imprimante, Accessoires, Autre)

3.4 Alertes critiques

  • Tickets critiques non résolus (rouge)
  • Visites prévues aujourd'hui (bleu)
  • Garanties expirées ou critiques (jaune)
  • Stock bas / accessoires sous seuil minimum (orange)

3.5 Tâches & Rappels

  • Créez des tâches avec titre, date d'échéance et technicien assigné
  • Les tâches en retard sont surlignées en rouge
  • Marquez une tâche comme terminée en cliquant sur le cercle

4. Gestion des Équipements

4.1 Liste des équipements

  • Affichage en tableau avec colonnes : type, nom, marque/modèle, IP, localisation, statut, technicien, actions
  • Filtres : par type (6 types), statut (5 statuts), site (multi-sélection)
  • Recherche : champ de recherche en temps réel
  • Préréglages de filtres : sauvegardez et rechargez vos configurations de filtres
  • Pagination : 50 équipements par page

4.2 Ajout d'un équipement

Cliquez sur Nouveau dans la barre d'outils. Remplissez les champs :

  • Champs obligatoires : Nom, Type
  • Champs recommandés : Marque, Modèle, Statut, Site, Localisation, Service
  • Champs optionnels : N° de série, IP, Date d'achat, Garantie, Quantité min., Fournisseur

4.3 Modification / Suppression

  • Cliquez sur une ligne pour ouvrir le panneau de détails
  • Utilisez les boutons Modifier ou Supprimer dans les actions de ligne
  • La suppression est logique (corbeille) — possibilité de restaurer

4.4 Détails d'un équipement

Le panneau de détails affiche :

  • Toutes les informations de l'équipement
  • Notes internes : ajoutez et consultez des notes
  • Historique des transferts
  • Documents : téléchargez et gérez les documents attachés

4.5 Actions groupées

  • Cochez plusieurs équipements avec les cases à cocher
  • Actions disponibles : Changement de statut, Transfert vers un site, Suppression groupée

4.6 Comparaison d'équipements

  • Sélectionnez 2 ou 3 équipements
  • Cliquez sur Comparer dans la barre flottante
  • Les différences sont surlignées en bleu

4.7 Import / Export

  • Import : depuis Excel (.xlsx) ou CSV — téléchargez le modèle vierge
  • Export : CSV, Excel (.xlsx) ou PDF

4.8 Statuts des équipements

5. Maintenance & Tickets

5.1 Module Maintenance

Le module de maintenance permet de gérer l'ensemble des tickets d'intervention.

5.2 Création d'un ticket

  • Cliquez sur Nouveau ticket
  • Sélectionnez le type : Maintenance ou Assistance
  • Remplissez : équipement concerné, description, priorité
  • Assigne éventuellement un technicien

5.3 Vues disponibles

  • Liste : tableau avec filtres par statut, priorité, type
  • Kanban : colonnes par statut (En attente → Ouvert → En cours → Résolu)
  • Rapport : statistiques détaillées exportables en Excel

5.4 Cycle de vie d'un ticket

  1. En attente — Ticket créé, en attente de prise en charge
  2. Ouvert — Prise en charge par un technicien
  3. En cours — Intervention en cours
  4. Résolu — Problème résolu (avec confirmation utilisateur + technicien)

5.5 Priorités

PrioritéDélai indicatif
Faible5 jours ouvrés
Normale3 jours ouvrés
Haute24 heures
Critique4 heures

5.6 Statistiques

  • Barre de statistiques en haut : tickets par statut
  • Rapport complet avec répartition par technicien, équipement, priorité
  • MTTR (Mean Time To Resolution) en heures
  • Export Excel multi-feuilles

6. Garanties

6.1 Vue d'ensemble

  • Cartes récapitulatives : Total suivis, Expirée, Critique, Bientôt fin, Non renseignées
  • Filtre par risque : Tous / Expirées / Critiques / Alerte / OK / Non renseignées
  • Tableau par site avec exposition au risque

6.2 États de garantie

ÉtatSignification
OKGarantie valide (> 90 jours restants)
AlerteExpire bientôt (≤ 90 jours)
CritiqueExpire très bientôt (≤ 30 jours)
ExpiréeGarantie terminée
Non renseignéeDate de garantie non fournie

6.3 Renouvellement de garantie

  • Module dédié "Renouv. Garantie" dans la navigation
  • Sélectionnez l'équipement et mettez à jour sa date de garantie

7. Visites de Site

7.1 Gestion des visites

  • Planifiez une visite : site, technicien, date, heure, objet
  • Associez une maintenance préventive à la visite
  • Statuts : Planifié → En cours → Terminé / Annulé / Reporté

7.2 Calendrier des visites

  • Vue mensuelle avec code couleur par statut
  • Cliquez sur un jour pour voir la liste des visites
  • Navigation mois par mois

7.3 Actions rapides

  • Marquer comme terminée (avec commentaire de validation)
  • Annuler ou reporter une visite

8. Transferts d'Équipements

8.1 Transfert individuel

  • Depuis la fiche équipement ou le menu Transferts
  • Remplissez : site de destination, localisation, service, motif, technicien
  • L'historique est conservé dans la fiche équipement

8.2 Transfert groupé

  • Sélectionnez plusieurs équipements dans la liste
  • Cliquez sur Transfert dans la barre d'actions groupées
  • Choisissez le site de destination

9. Licences Logicielles

9.1 Gestion des licences

  • Ajoutez une licence : nom, éditeur, clé de licence, nombre de sièges
  • Suivi des sièges utilisés avec barre de progression visuelle
  • Dates d'achat et d'expiration
  • Liez un fournisseur et un équipement

9.2 Indicateurs

  • Rouge : Sièges épuisés (≥ 100%)
  • Orange : Capacité critique (≥ 80%)
  • Vert : Capacité normale
  • Badge "Expirée" ou "Bientôt" sur les cartes

10. Demandes d'Achat

10.1 Création

  • Cliquez sur Nouvelle demande
  • Remplissez : titre, type d'équipement, quantité, coût estimé, priorité, justification, service fournisseur

10.2 Workflow d'approbation

  • Statut En attente → en attente de décision
  • Les administrateurs peuvent Approuver ou Rejeter (avec motif)
  • Les demandes sont regroupées par statut dans l'interface

11. Retours Garantie (RMA)

11.1 Gestion des RMA

  • Créez une demande RMA : équipement, n° de série, vendeur, motif, n° RMA
  • Suivez les dates d'expédition et de réception
  • Statuts : Ouvert → Expédié → Résolu
  • Liez un fournisseur et un technicien

12. Détection d'Anomalies

12.1 Anomalies automatiques

  • Le système détecte automatiquement les équipements avec 3 pannes ou plus dans les 6 derniers mois
  • Affichage : nom, type, service, localisation, nombre de tickets
  • Lien direct vers le module Maintenance pour intervention

13. Contrats de Maintenance

13.1 Gestion des contrats

  • Créez des contrats : titre, fournisseur, n° contrat, site, dates début/fin, montant
  • Indicateurs d'expiration (expiré / expire bientôt)
  • Badge de statut : Actif / Inactif

14. Fournisseurs

14.1 Gestion (Administrateur)

  • Liste des fournisseurs avec coordonnées
  • Création, modification, suppression
  • Vue consolidée : équipements, contrats, achats, licences, RMA liés

15. Gestion des Sites

15.1 Sites (Administrateur)

  • Créez et gérez les sites (nom, ville, pays, adresse, coordonnées GPS)
  • Chaque site affiche le nombre d'équipements associés
  • Sélecteur de site dans le dashboard (filtre multi-sélection)

15.2 Carte interactive

  • Visualisez tous les sites géolocalisés sur une carte Leaflet
  • Marqueurs cliquables avec informations du site

16. Utilisateurs & Permissions

16.1 Gestion (Administrateur)

  • Créez, modifiez, supprimez des comptes utilisateurs
  • Attribuez un rôle : Administrateur, Technicien, Utilisateur
  • Définissez les permissions et les sites autorisés
  • Bloquez / débloquez un compte

16.2 Rôles

RôleAccès
AdministrateurAccès complet à toutes les fonctionnalités
TechnicienPermissions configurables (lecture, écriture, modification)
UtilisateurPermissions configurables (lecture, écriture, modification)

16.3 Permissions

PermissionDroits
LectureConsulter la liste des équipements
ÉcritureAjouter de nouveaux équipements
ModificationModifier et supprimer des équipements

17. Messagerie

17.1 Conversations

  • Globale : visible par tous les utilisateurs
  • Groupes : conversations privées créées par les administrateurs
  • Messages directs (DM) : conversations privées entre deux utilisateurs

17.2 Utilisation

  • Cliquez sur l'icône de messagerie (💬) dans la barre d'actions
  • Sélectionnez une conversation dans la liste de gauche
  • Saisissez votre message et appuyez sur Envoyer
  • Les messages non lus sont signalés par un badge rouge

18. Centre de Notifications

  • Cliquez sur la cloche 🔔 dans la barre d'actions
  • Aperçu rapide des notifications : tickets critiques, garanties expirées, visites du jour, stock bas
  • Cliquez sur Voir tout pour le centre complet
  • Marquez comme lu (une ou toutes les notifications)
  • Types : Info (bleu), Avertissement (orange), Erreur (rouge)

20. Calendrier Unifié

  • Vue mensuelle avec tous les événements (maintenances et visites)
  • Légende : Rouge = Maintenance, Bleu = Visites
  • Cliquez sur un jour pour voir la liste des événements
  • Navigation mois par mois / année par année

21. Carte des Sites

  • Visualisation de tous les sites sur une carte interactive (Leaflet)
  • Marqueurs positionnés selon les coordonnées GPS
  • Cliquez sur un marqueur pour voir les informations du site

22. Inventaire Physique

  • Module guidé pour réaliser un inventaire physique
  • Barre de progression (% scanné)
  • Bouton à cocher pour chaque équipement scanné
  • Les équipements scannés passent en vert avec texte barré
  • Bouton Voir manquants pour afficher les équipements non scannés
  • Intégration des résultats Ping (en ligne/hors ligne)

23. Monitoring Temps Réel

23.1 Sessions actives

  • Liste des utilisateurs connectés (en ligne / inactif)
  • Date/heure de connexion, dernière activité, adresse IP
  • Badges de rôle

23.2 Activité en direct

  • Flux des actions utilisateurs en temps réel
  • Actualisation automatique toutes les 30 secondes
  • Bouton d'actualisation manuelle

24. Journal d'Activité

  • Registre complet de toutes les actions utilisateur
  • Filtres : par utilisateur, type d'action, période
  • Types tracés : Connexion, Déconnexion, Ajout/Modification/Suppression, Transfert, Réforme, Export
  • Export Excel ou PDF
  • Barre de statistiques : total actions, utilisateurs actifs, connexions

25. Dashboard TV

  • Affichage plein écran optimisé pour écran large / projecteur
  • 8 tuiles KPI : Total, Actifs, Défaillants, Tickets ouverts, Critiques, Garanties exp., Visites planif., Non visités
  • Indicateur "Live" avec point vert clignotant
  • Date et heure actuelles
  • Bannière d'alerte rouge clignotante si tickets critiques
  • Fermeture avec le bouton X

26. Recherche Globale

  • Raccourci : Ctrl+K
  • Recherche dans : équipements (nom, marque, modèle, n° série, IP, localisation, service), sites (nom, ville, pays)
  • Navigation directe vers l'élément trouvé

27. QR Code & Barcode

27.1 QR Code

  • Chaque équipement peut afficher son QR Code
  • Options : visualiser, télécharger, imprimer

27.2 Barcode (Code-barres)

  • Génération au format CODE128
  • Téléchargement en PNG
  • Impression directe

27.3 Scanner QR

  • Utilisez la caméra de l'appareil pour scanner les QR Codes des équipements
  • Navigation directe vers l'équipement scanné

28. Paramètres

28.1 Webhook

  • Configurez une URL Slack ou Teams pour les notifications
  • Bouton de test pour valider la configuration

28.2 Masquage des données sensibles

  • Activez/désactivez le masquage des n° de série, adresses IP, clés de licence
  • Bascule rapide depuis la barre d'actions (icône œil)

28.3 Rapport RGPD

  • Générez un PDF de conformité RGPD listant les catégories de données collectées

28.4 Sauvegarde

  • Téléchargez une sauvegarde complète de la base de données au format JSON
  • Réservé aux administrateurs
